#Morphink
MorphInk is a React based AI sketch studio designed for drawing, sketch cleanup, and artwork export. It combines a modern studio style interface with a canvas workspace, layered editing, gallery save/load
, and account based registration.
 ##Features

- Upload sketches and convert them into editable canvas art
- Clean Line Art generation from sketch input
- Auto Color simulation for quick color effects
- Drawing tools: pencil, brush, marker, spray, shape, line, rectangle, sticker, and eraser
- Canvas resizing with width and height controls
- Layer control for opacity and visibility
- Save to gallery and load saved images
- Export artwork as PNG or SVG
- User account system  with register/login
